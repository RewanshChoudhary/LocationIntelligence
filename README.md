A real-time platform for ingesting, processing, and visualizing IoT sensor data. Built with Spring Boot, Kafka, and PostgreSQL (PostGIS), it enables scalable data pipelines and live geospatial analytics through a React-based dashboard.

Features
Real-Time Data Streaming with Kafka producers and consumers

WebSocket Support for live updates to clients

Geospatial Data Storage using PostgreSQL + PostGIS

Interactive Dashboard with live charts and maps

Modular Architecture (producer, consumer, controller, service layers)

Docker Support for database and Kafka services
